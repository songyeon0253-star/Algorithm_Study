# Algorithm_Study

## Purpose
- 삼성 SW 역량테스트 AD 등급 취득을 위한 알고리즘 스터디 GitHub
---

## Members
- 구성원 : 구도연, 송가연, 이근화
---


## Directory Structure
```
{이름}/
├─ solved/
└─ unsolved/
```

## File Naming
```
BOJ_문제번호.py
SWEA_문제번호.py
PROG_Lv_n.py
```

## Commit Convention
```
[tag] [BOJ_문제번호] - 이름
[tag] [SWEA_문제번호] - 이름
[tag] [PROG_Lv_n] - 이름
```

## Commit Convention

```
[type] [BOJ-문제번호] 문제이름 - 이름
[type] [SWEA-문제번호] 문제이름 - 이름
```

| tag      | mean                     | explain                                              |
|-----------|--------------------------|---------------------------------------------------|
| solved    | 해결된 문제 | 문제를 해결했거나, 못했던 것을 이후에 해결했을 때 수정 |
| unsolved  | 해결되지 않은 문제                 | 문제 해결 실패시 사용                             |
| refactor  | 코드 로직 개선   | 외부 동작 변경 없이 내부 구조 개선해 가독성, 유지보수성, 확장성, 효율성 높인 견우사용          |        |                                                   |
